# Holberton AI4Devs

A comprehensive repository for AI-assisted software development, featuring prompt patterns for various development tasks and practical debugging exercises.

## Repository Overview

This repository contains resources for developers learning to leverage AI as a coding assistant, with a focus on effective prompting techniques and debugging methodologies.

## Folder Structure

### [prompt_patterns_library/](prompt_patterns_library/)
A curated collection of AI prompt patterns and best practices for software development tasks.

- **[prompt_use_cases.md](prompt_patterns_library/prompt_use_cases.md)** - Overview of different use cases for prompts
- **[prompts/](prompt_patterns_library/prompts/)** - Ready-to-use prompt templates organized by development category:
  - Architecture & Design: ADR, API design, scalability planning
  - Code Quality: Performance optimization, security hardening
  - Debugging: Incident mitigation, log interpretation, root cause analysis
  - Development: Refactoring, testing strategies

### [prompting_debug_assistant/](prompting_debug_assistant/)
Hands-on debugging exercises demonstrating AI-assisted bug discovery and resolution.

- **[bug_reports.md](prompting_debug_assistant/bug_reports.md)** - Summary of identified bugs
- **[reflection.md](prompting_debug_assistant/reflection.md)** - Analysis and insights from debugging exercises
- **[bug_snippets/](prompting_debug_assistant/bug_snippets/)** - Original buggy code samples in multiple languages (Python, JavaScript, Java, C++)
- **[bug_fixes/](prompting_debug_assistant/bug_fixes/)** - Corrected code with fixes applied and validation notes

## Getting Started

1. Explore [prompt_patterns_library/](prompt_patterns_library/) to learn effective prompting techniques
2. Review [prompting_debug_assistant/](prompting_debug_assistant/) to see practical debugging examples
3. Use the prompt templates and debugging strategies in your own AI-assisted development workflow
