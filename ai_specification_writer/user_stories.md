# CarPool Connect User Stories

## Core Employee Features

### User Story 1

**As a corporate employee**, I want to find colleagues with similar commute routes so that I can save money and drive less.

**Acceptance Criteria**:
- App shows coworkers within 2 miles of my route  
- Matches consider my work schedule (±30 minutes flexibility)  
- I see 3-5 best matches with car details and ratings  

**Priority**: MVP

### User Story 2
**As a corporate employee**, I want to join a carpool group so that I can coordinate regular rides with the same people.

**Acceptance Criteria**:
- I can accept a match and join the group instantly  
- Group chat opens automatically for coordination  
- My calendar gets updated with carpool schedule  

**Priority**: MVP

### User Story 3
**As a corporate employee**, I want to message my carpool group so that we can handle schedule changes easily.

**Acceptance Criteria**:
- Simple group chat works inside the app  
- Push notifications for urgent messages  
- Schedule changes automatically notify everyone  

**Priority**: High

## Corporate Management Features

### User Story 4
**As an HR manager**, I want to see carpool participation rates so that I can report progress to leadership.

**Acceptance Criteria**:
- Dashboard shows % of employees carpooling  
- Filter by department and location  
- Export monthly participation reports  

**Priority**: MVP

### User Story 5
**As a facility manager**, I want to reserve parking spots for carpoolers so that single drivers park farther out.

**Acceptance Criteria**:
- Verified carpool groups get preferred parking  
- App shows real-time parking availability  
- Daily parking assignments update automatically  

**Priority**: High

### User Story 6
**As a sustainability officer**, I want CO2 savings reports so that I can prove environmental impact to executives.

**Acceptance Criteria**:
- Dashboard shows tons of CO2 saved monthly  
- Compare against company baseline emissions  
- Generate PDF reports with charts for presentations  

**Priority**: MVP

## Integration & Admin Features

### User Story 7
**As an IT administrator**, I want employees to login with company accounts so that no one needs new passwords.

**Acceptance Criteria**:
- Single sign-on works with Okta/Azure AD  
- New hires automatically get access  
- Departing employees lose access immediately  

**Priority**: MVP

### User Story 8
**As a facility manager**, I want parking usage analytics so that I can plan infrastructure better.

**Acceptance Criteria**:
- See parking space utilization before/after carpooling  
- Track peak usage times and problem areas  
- Monthly summary shows parking spaces freed up  

**Priority**: Medium

### User Story 9
**As a corporate employee**, I want to see my personal savings so that I stay motivated to carpool.

**Acceptance Criteria**:
- App calculates money saved vs driving alone  
- Shows monthly and yearly totals  
- Compares with public transit/parking costs  

**Priority**: High

### User Story 10
**As an HR manager**, I want employee satisfaction feedback so that I can improve the program.

**Acceptance Criteria**:
- Monthly survey asks about carpool experience  
- Anonymous feedback with optional comments  
- Dashboard shows satisfaction trends over time  

**Priority**: Medium

### User Story 11
**As a corporate employee**, I want calendar integration so that schedule changes automatically update my carpool.

**Acceptance Criteria**:
- Connects with Outlook/Google Calendar  
- Schedule conflicts trigger group notifications  
- Suggests backup carpools when I'm unavailable  

**Priority**: High

### User Story 12
**As a sustainability officer**, I want industry benchmarking so that I can show leadership we're ahead of competitors.

**Acceptance Criteria**:
- Compare our CO2 reduction vs industry averages  
- Show parking optimization vs similar companies  
- Annual report with competitive positioning  

**Priority**: Low
