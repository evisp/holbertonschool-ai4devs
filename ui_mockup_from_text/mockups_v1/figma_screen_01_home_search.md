# figma_screen_01_home_search.md

Design the complete interface for "Home/Search Screen" in the context of "Property Pulse Albania".

Task:
Create the landing and search screen for a real estate platform.

Context:
Users arrive here to begin searching for homes in Albania and quickly understand what the product does.

Elements:
Top navigation, logo text "Property Pulse Albania", hero section, prominent search bar, quick filters, featured listings preview, short trust section, clean footer.

Behavior:
The main call to action is to start searching properties by city, price, and property type. The page should guide users directly into action without too much text.

Constraints:
Desktop web layout, responsive structure, modern and professional, light theme, blue or blue-green accent, clean cards, accessible text contrast, realistic real estate content, no lorem ipsum, use Albanian cities like Tirana, Durrës, Vlorë, Shkodër.
