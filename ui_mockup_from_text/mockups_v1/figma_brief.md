# figma_brief.md

## Product
Property Pulse Albania

## Product Summary
A real estate web platform for Albania that helps buyers, sellers, and agents explore property listings, view homes on a map, compare options, and understand whether a listing price looks low, fair, or high based on similar properties.

## Main Users
- Buyers: search homes and judge whether the asking price is reasonable.
- Sellers: understand how their property compares with the market.
- Agents: help clients browse, compare, and evaluate listings.

## MVP Goal
Design the first draft of a clean, modern, trustworthy web interface for the main property-search workflow.

## Core Screens
- Home/Search Screen
- Map + Listings Screen
- Property Details Screen

## Optional Later Screens
- Compare Properties Screen
- Price Predictor Screen

## Brand Direction
- Professional, modern, simple, trustworthy
- Real estate focused
- Clean layouts with strong content hierarchy
- Light theme
- Main accent in blue or blue-green
- Easy to scan cards
- Strong visual emphasis on price, location, and key features

## UX Priorities
- Make search easy
- Make map exploration clear
- Make price evaluation understandable
- Keep the interface mobile-friendly
- Use real-looking real estate content, not vague placeholders

## Accessibility
- High contrast text
- Clear buttons and form fields
- Visible focus states
- Easy-to-read typography
- Touch-friendly controls
