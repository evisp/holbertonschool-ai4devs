# figma_screen_03_property_details.md

Design the complete interface for "Property Details Screen" in the context of "Property Pulse Albania".

Task:
Create the detailed property page for a selected home.

Context:
Users open this page after choosing a listing from search results or the map. They want to understand the home and whether the asking price is reasonable.

Elements:
Property image gallery, title, price, location, key facts, pricing status badge, estimated market value panel, main features list, description, mini map, similar properties section, compare button, contact button.

Behavior:
The page should make decision-making easier. The most important items are asking price, estimated value, property facts, and location. The estimated value section should feel informative and simple, not technical.

Constraints:
Desktop web page, responsive structure, light theme, trustworthy and clean, strong content hierarchy, accessible contrast, realistic real estate content, avoid technical jargon, make pricing insight very clear.
