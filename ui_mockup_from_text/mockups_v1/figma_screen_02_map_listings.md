# figma_screen_02_map_listings.md

Design the complete interface for "Map + Listings Screen" in the context of "Property Pulse Albania".

Task:
Create the main browsing screen where users explore properties on a map and in a listings panel.

Context:
This is the core working screen of the product. Users search, filter, scan listings, and explore location visually.

Elements:
Header, search summary bar, filter sidebar or top filter row, interactive map area, property markers, listing cards, sorting controls, pricing badges, pagination or load more pattern.

Behavior:
Users should be able to understand the layout immediately. The map and listings should feel balanced. Property cards should highlight price, location, size, rooms, and a small label such as Underpriced, Fair Price, or Overpriced.

Constraints:
Desktop-first responsive web screen, split layout with clear visual hierarchy, light theme, professional styling, high readability, realistic property data, no clutter, no unnecessary charts, no luxury visual style.
