# UI Concept - Property Pulse Albania

## Purpose
Help users explore Albanian property listings, understand market prices, and compare homes more easily. The interface should make property search simple, visual, and fast, while also helping users judge whether a listing looks like a good deal.

## Users
- Buyers: search for homes, compare options, and understand whether a property is fairly priced.
- Sellers and agents: review listings, compare similar properties, and check pricing position.
- Students and instructors: use the platform as a practical example of data, product design, and web application development.

## Core Screens
- Home page with search bar, quick filters, and featured listings.
- Listings page with filters, sorting, and a grid or list of properties.
- Map view page showing properties by location with clickable property cards.
- Property details page with price, size, location, features, and estimated market value.
- Comparison page for viewing two properties side by side.
- Price predictor page where users enter property details and get an estimated price.
- Optional chat page for simple property questions and suggestions.

## Actions
- Search properties by city, neighborhood, price, size, and property type.
- Filter and sort listings based on user preferences.
- View properties on a map and open summary cards.
- Open a full property details page from a listing or map card.
- Compare two properties side by side.
- Enter property details to get a market price estimate.
- Check whether a listing is underpriced, fairly priced, or overpriced.
- Ask simple questions through the optional chat assistant.
